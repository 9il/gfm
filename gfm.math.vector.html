Implements swizzling.

Example:
vec4i vi = [4, 1, 83, 10];
assert(vi.zxxyw == [83, 4, 4, 1, 10]);

pure void  opDispatch(string op, U)(U x) if (op.length >= 2 && isValidSwizzleUnique!op && is(typeof(Vector!(T, op.length)(x))));

Support swizzling assignment like in shader languages.

Example:
vec3f v = [0, 1, 2];
v.yz = v.zx;
assert(v == [0, 2, 0]);
