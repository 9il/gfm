Provides the Host, Server, and Client classes.

class  Client: gfm.enet.host.Host;

A subclass of Host that exists to concisely create a client-oriented Host

The Host superclass' address is set to null which disallows other peers from connecting to the Host.

class  Server: gfm.enet.host.Host;

A subclass of Host that exists to concisely create a server-oriented Host

The Host superclass' address is set to ENET_HOST_ANY which binds the server to the default localhost.
