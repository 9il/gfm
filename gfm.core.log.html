abstract class  Log;

Provides a common logging interface for GFM.

Known Bugs
 Log is not thread-safe. Messages will get squashed when output by multiple threads.
Deprecated
This whole module will go away when std.logger is there.

enum  MessageType: int;

Log message severities.

 DEBUG

This level is intended for debugging and should not happen in production.

 INFO

This level is for informational messages.

 WARNING

An error which is not fatal and was recovered.

 ERROR

A serious error that can't be recovered.
